Components of a Game

When developing a game there are many parts of it that need to be taken care of. Those individual components of the game will be combined together in order to make an end product that will be able to be tested by other people in the company. Some parts of a game are:

  • Game world (the place where the game is set to be in)
  • Art (Visuals)
  • Sound (Audio)

This shows the art from the game Hollow Knight and the game world from Monster Hunter World.

Technology Used

Creating games not only requires the knowledge, but they also require certain tools so that they serve as foundations where the game can be built upon. Some of the tools that are used to build games include game engines and sound software. There are also some game design tools that contain a game engine and sound development such as Unity and Unreal Engine.
